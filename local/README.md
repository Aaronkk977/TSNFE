# Local Files

This folder stores machine-specific or private helper files that should not clutter the repository root.

## Files

- `analyst_list.txt`: personal analyst/channel notes.
- `conda_commands.sh`: local environment command cheatsheet.
- `cookies.txt`: browser/exported cookies for yt-dlp or web automation.

## Security

- Keep API keys and cookies out of Git history.
- `local/cookies.txt` is ignored by `.gitignore`.
- Rotate credentials immediately if they were ever committed.
